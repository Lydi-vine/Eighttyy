 > "".endsWith;

The endsWith() method in JavaScript is used to check if a string ends with 
a specified substring. It returns true if the string ends with the specified 
characters, otherwise false. It is case sensitive.

Syntax: str.endsWith(searchString[, length])

Example:
	let sentence = "Hello, World!";
	console.log(sentence.endsWith("World!")); // true
	console.log(sentence.endsWith("world!")); // false
	let subSentence = "Hello, World!";
	console.log(subSentence.endsWith("World", 12)); // true
